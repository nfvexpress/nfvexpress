Install NFV Express

Preview OpenStack® on FreeBSD® today

Requirements

  • Fresh installed Non-UEFI FreeBSD® 11 server on x86-64 hardware with 8GB RAM
  • EPT and IOMMU processor features to use Xen® hypervisor
  • Configured Xen® Dom0
  • At least 1 NIC with Static IP

Single node installation using script

# git clone https://github.com/nfvexpress/nfvexpress-install
# single-node.sh xen openstack 192.168.1.30 re0 tap0
...
OpenStack Dashboard has been installed
Navigate to http://192.168.1.30/
Domain: default
Username: admin
Password: ADMIN_PASS

About

Few words about the project

NFV Express is an Open Source R&D project aimed to build a full-featured scalable and secure platform for Cloud Computing on FreeBSD.

The project was started in October, 2016 and today it became possible to install most of core OpenStack® services and run them entirely on FreeBSD®!
